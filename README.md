## Requirements

To run this project locally you need the following installed:

* NodeJS 8 or higher
* Yarn

## Installation

To install all dependencies, run

```sh
yarn
```

To run the local development environment, run

```sh
yarn run dev
```

To build all dependencies, run

```sh
rm -rf dist && yarn run build
```
